## 常见问题

> * 系统事件有哪些
>
>   答:utils/event/目录下包含了所有事件头文件, 分别有蓝牙,WIFI,按键,设备的事件
>
> * 用户如何最简单的处理事件
>
>      答:在对应app的event_handler处理事件
